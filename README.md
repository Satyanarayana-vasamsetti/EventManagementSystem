# EventManagementSystem
A web-based Event Management System built using JDBC, JSP, Servlets, and MySQL to streamline event planning, registration, and management.

📌 Features
✅ User authentication (Admin & Participants)
✅ Event creation, modification, and deletion (Admin)
✅ Event registration and participant management
✅ Dynamic event listings with details
✅ Database-driven architecture using MySQL

🛠️ Technologies Used
Frontend: JSP, HTML, CSS, Bootstrap
Backend: Servlets, JDBC
Database: MySQL
Server: Tomcat
